# 🌾 Krishi-Sahayak AI

**Your Smart Farming Assistant — Voice-Enabled Agricultural Advisor**

Built as part of a hackathon project, Krishi-Sahayak AI is a multilingual, voice-first farming assistant designed to help farmers get instant advice on crop health, weather, and market prices — in **English, Hindi, and Punjabi**.

 **[Live Demo](https://vishesharma20.github.io/krishi-sahayak-ai/)**

---

## Features

- 🎤 **Voice Assistant** — Ask farming questions using speech (powered by the Web Speech API)
- 🌐 **Multilingual Support** — Switch seamlessly between English, हिंदी, and ਪੰਜਾਬੀ
- 💬 **Text-based Q&A** — Type your farming question and get instant AI-style advice
- 📷 **Crop Disease Detection (Demo)** — Upload a photo of your crop for a simulated disease analysis
- 🔊 **Text-to-Speech** — Have the AI's response read aloud in your selected language
- 🌤️ **Weather & Market Price Widgets** — Quick localized farming info at a glance

## Tech Stack

- **HTML5** — Structure
- **CSS3** — Styling and responsive layout
- **JavaScript (Vanilla)** — App logic, DOM manipulation
- **Web Speech API** — Speech Recognition (voice input) and Speech Synthesis (voice output)

No frameworks, no backend — runs entirely in the browser.

---

## Getting Started Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/vishesharma20/krishi-sahayak-ai.git
   ```
2. Open the folder and launch `index.html` in your browser (Chrome recommended for full Web Speech API support).

That's it — no build steps or dependencies required.

---

## 🏆 About This Project

This project was built for a hackathon to explore how voice and AI-driven interfaces can make agricultural guidance more accessible to farmers who may not be comfortable with typing or English-only tools.
